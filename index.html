Renkler, a CSS Framework

Renkler is a CSS Framework, it has been created using SCSS and JS. It provides ready to use classes for your project. It contains utility classes and proper color theme for the components.

How to use?

1) Copy the follwing line inside the Head element.

<link rel="stylesheet" href=" https://cdn.jsdelivr.net/gh/Munawar-git/Renkler@main/scss/style.css" />

2) Copy the follwing line at the end of Body element.

<script type="module" src=" https://cdn.jsdelivr.net/gh/Munawar-git/Renkler@main/index.js" ></script>

Utility Classes

Margins

List of Classes


General class format:

m<side-suffix>-<thickness-suffix>


Side Suffixes:

  • No side suffix, it means all sides
  • t : for Top only
  • b : for Botton only
  • l : for Left only
  • r : for Right only
  • y : for Top and Bottom only
  • x : for Left and right only

Thickness Suffixes:

  • 0 : no margin
  • 1 : 1rem
  • 2 : 2rem
  • 3 : 4rem
  • 4 : 6rem
  • 5 : 8rem
  • auto : only with 'x' and 'y'

Example:

Any combination of sizes and suffixes are available to use to set margin.


The followings are some example

  • m-5 : margin on all sides thickness 5
  • mt-4 : margin on top with thickness 4
  • mb-2 : margin on bottom with thickness 2
  • ml-3 : margin on left with thickness 3
  • mr-5 : margin on right with thickness 5
  • mx-1 : margin on left and right with thickness
  • my-auto : margin on top and bottom with thickness of auto

Usage


<div class="box bg-red mx-auto"></div>

<div class="box bg-red ml-3"></div>

<div class="box bg-red m-2"></div>

<div class="box bg-red mb-1"></div>

Grid System

List of Classes


For Grid their are three main type of classes.

  • container
  • row
  • column

Container:

The container class put the content it the middle of the screen. It has the max width of 1200px, if the width is less then that then it will take the 100% of screen width.


Row:

The row class is the base of grid system. The HTML Tag having class row contains the grid structure, the grid will not work outside row.


Column:

The column class is the grid element. It is always used inside the row. A row can contain maximum of 12 columns. The column classes also contain classes that are responsive to the screen size. If the total sum of columns in a row exceeds 12, then the exceding columns will be moved to the second row.


General class format:

col-<size-suffix>-<number-of-cols>;


Side Suffixes:

  • No side suffix, it means all screen sizes
  • xs : 0px and above
  • sm : 480px and above
  • md : 720px and above
  • lg : 960px and above
  • xl : 1200px and above

Number of Columns:

  • 1 to 12

Example:

Any combination of number of cols and size suffixes are available to use.


The followings are some example

  • col-8 :occupies 8 columns in every screen size
  • col-xs-5 :occupies 5 columns in xs screen size
  • col-sm-4 :occupies 4 columns in sm screen size
  • col-md-3 :occupies 3 columns in md screen size
  • col-lg-2 :occupies 2 columns in lg screen size
  • col-xl-1 :occupies 1 columns in xl screen size

Usage


6 boxes each with col-2 class.


<div class=" row">
<div class="bd-2-blue bg-dark-blue col-2"></div>
</div>

Description of the List of Classes


Top Most Parent Classes:

navbar, navbar-<color>


Add "navbar-fixed" Class to Parent Element to fix the Navbar at the Top.

Example:

<nav class="navbar-yellow navbar-fixed> shadow mb-2">


List of colors

For differet colors of navbar use the following color scheme


  • primary
  • secondary
  • purple
  • blue
  • green
  • yellow
  • orange
  • red
  • black

Example:

<nav >class="navbar-COLOR shadow mb-2">
<nav >class="navbar-blue shadow mb-2">

Navbar Items

Inside the "ul" element with class "nav-list"span>, add the following HTML lines to add more items to the Navbar.


<li class="nav-item">
  <a href="" class="nav-link">ITEM_NAME</a>
</li>

Add the correct number of items in style="--nav-items: NUMBER_OF_NAV_ITEMS".


Example:

<ul class="nav-list" name="collapse" style="--nav-items: 5">

Navbar Toggle

The "for" attribute of label element with class toggle-btn should be the same as "name" attribute of ul element with class nav-list


When using multiple Navbars, for each Navbar the "for" and "name" attribute pair should be unique.


If multiple Navbars have same "for" and "name" attributes pair, then the Navbars will malfunction


Example:

In the following example, the "for" and "name " attribute has same names.


<label class="toggle-btn" for="collapse">
  <div class="nav-btn"></div>
</label>
<ul class="nav-list" name="collapse"style="--nav-items: 5">

Usage


To create a Navbar, copy and paste the following code to your HTML document and then edit it accordingly.


<nav class="navbar-yellow shadow mb-2">
 <div class="nav-title">
  <a href="">Renkler</a>
 </div>
 <label class="toggle-btn" for="collapse">
  <div class="nav-btn"></div>
 </label>
 <ul class="nav-list" name="collapse" style="--nav-items: 5">
  <li class="nav-item">
   <a href="" class="nav-link">Home</a>
  </li>
  <li class="nav-item">
   <a href="" class="nav-link">About</a>
